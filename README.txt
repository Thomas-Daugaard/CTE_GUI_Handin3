How to use: 

1) Start the API. 
2) Start the application by starting a command prompt from the "ModelsManagement" project and run:"npm run serve". 
3) Copy the IP address to your browser. 

4) Go to the login and enter the manager login.
5) Try out the different actions/tabs. 

6) logout and login with a model login. 
7) Try out the different actions/tabs.  


Enjoy!
